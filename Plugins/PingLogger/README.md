# PingLogger
version 1.0.9
# Features
  * Logs any messages in which you are mentioned
  * Tap to sender name to copy his ID
  * Up to 50 logged messages
  * Stays persist after reloading Discord
  * Support Multi-language (only RU, EN; ***WIP***)
